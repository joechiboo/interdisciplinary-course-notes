# 1141_0295_跨領域創新科技應用

**課程期間**: 2025/08/01 ~ 2026/01/31
**必修**: 否

## 教學目標

### 一、認知
1. 瞭解當前主流與未來5-10年創新科技的理論與應用
2. 瞭解跨領域教學設計的原理與方法
3. 瞭解科技融入教學的多元模式與最佳實踐

### 二、技能
1. 具備運用創新科技解決教育問題的能力
2. 具備設計與實施跨領域科技融入教學的能力
3. 具備評估科技應用成效與持續改進的能力

### 三、情意
1. 培養創新思維與跨域合作的態度
2. 增進對科技融入教育的反思與批判能力
3. 發展專業成長與終身學習的熱忱

## 課程大綱

### 第1週：課程導論與創新科技概覽
- 課程介紹：目標、評量方式與學習期望
- 創新科技的定義與發展脈絡
- 當前教育科技趨勢與未來展望
- 跨領域思維與創新教學的關聯
- 小組活動：教學現場科技應用現況分享與挑戰討論
- 建立學習社群：依據不同專長與興趣組成跨領域學習小組

### 第2週：設計思考與教育創新
- 設計思考的核心原則與流程
- 以學習者為中心的教育創新方法
- 案例分析：成功的教育創新實例
- 工具介紹：設計思考工具包與應用方法
- 實作：運用設計思考方法定義教學現場的挑戰
- 小組討論：從教學實務中發掘創新機會

### 第3週：人工智慧在教育中的應用基礎
- 人工智慧的核心概念與發展現況
- 生成式AI工具在教育中的應用與限制
- 案例分析：AI輔助教學設計與個人化學習
- 工具實作：體驗並評估教育適用的AI工具
- 倫理討論：AI在教學中的倫理考量與最佳實踐
- 小組活動：設計AI輔助的教學活動雛形

### 第4週：擴增實境與虛擬實境在教學中的應用
- AR/VR/MR技術原理與教育潛力
- 沉浸式學習體驗的設計原則
- 案例分析：國小各學科的AR/VR應用實例
- 工具實作：低門檻AR應用開發（無需程式基礎）
- 教學設計：規劃AR/VR融入課程的策略
- 小組討論：評估AR/VR在不同學科的適用性

### 第5週：物聯網與智慧教室
- 物聯網技術基礎與教育應用
- 智慧教室的設計與實施策略
- 案例分析：智慧校園與智慧教室實踐案例
- 工具實作：簡易物聯網教學應用（使用現成平台）
- 教學設計：物聯網融入STEAM教育的可能性
- 小組活動：設計低成本智慧教室方案

### 第6週：程式思維與運算思考教學
- 運算思維的核心概念與教學策略
- 程式教育在國小各年段的實施方法
- 案例分析：無螢幕與實體程式教學活動
- 工具實作：圖形化程式設計工具與教學應用
- 跨領域整合：程式思維融入各學科教學
- 小組討論：設計適合不同年齡層的運算思維活動

### 第7週：數位敘事與多媒體創作
- 數位敘事的教育價值與應用方法
- 多媒體創作工具與教學策略
- 案例分析：學生數位作品集與創作平台
- 工具實作：數位敘事工具與多媒體製作
- 教學設計：將數位敘事融入語文與社會學習
- 小組活動：設計跨領域數位敘事教學單元

### 第8週：期中專案提案與回饋
- 學生展示期中專案提案（跨領域科技融入教學設計）
- 同儕評論與建設性回饋
- 專家指導與改進建議
- 實施計畫調整與精進
- 反思討論：創新應用的可行性與挑戰
- 下階段專案發展規劃

### 第9週：數據素養與教育數據分析
- 教育數據的類型與收集方法
- 數據素養的核心概念與教學策略
- 案例分析：以數據驅動的教學決策
- 工具實作：教育數據視覺化與分析工具
- 教學設計：數據素養融入各學科教學
- 小組討論：如何培養學生的數據思維能力

### 第10週：遊戲化學習與教育遊戲設計
- 遊戲化學習的理論基礎與設計原則
- 教育遊戲的類型與評估標準
- 案例分析：成功的教育遊戲與遊戲化課程
- 工具實作：低門檻教育遊戲設計平台
- 教學設計：遊戲化元素融入教學活動
- 小組活動：設計跨領域遊戲化學習單元

### 第11週：創客教育與設計實作
- 創客教育的理念與實施策略
- 設計實作在培養創新思維中的角色
- 案例分析：國小創客教育成功案例
- 工具實作：低成本創客工具與材料運用
- 教學設計：將創客精神融入各學科教學
- 小組討論：規劃校園創客空間與活動

### 第12週：數位評量與學習分析
- 數位評量的類型與設計原則
- 學習分析的基本概念與應用方法
- 案例分析：創新評量與即時回饋系統
- 工具實作：數位評量工具與學習儀表板
- 教學設計：整合形成性評量的教學活動
- 小組活動：設計多元數位評量方案

### 第13週：未來教室與創新學習空間
- 未來教室的設計理念與趨勢
- 學習空間對教學模式的影響
- 案例分析：創新學習空間的實踐案例
- 工具實作：學習空間規劃與模擬工具
- 教學設計：適應不同學習空間的教學策略
- 小組討論：重新想像與設計校園學習空間

### 第14週：新興科技與未來教育趨勢
- 新興科技概覽：腦機介面、量子計算、元宇宙等
- 未來5-10年可能影響教育的科技趨勢
- 案例分析：前瞻性教育科技應用實驗
- 專家對談：邀請產業專家分享前瞻視野
- 反思討論：科技變革中的教師角色轉變
- 小組活動：設計未來教育情境與應對策略

### 第15週：專案實作工作坊
- 期末專案開發與實作指導
- 跨領域合作與資源整合策略
- 技術支援與問題解決
- 專案管理與時程規劃
- 成果展示準備與呈現技巧
- 同儕互助與專家諮詢

### 第16週：期末專案展示與課程總結
- 學生展示期末專案成果
- 專家評論與建設性回饋
- 實施成效分享與反思
- 未來應用與持續發展規劃
- 課程內容回顧與統整
- 建立持續專業發展社群
